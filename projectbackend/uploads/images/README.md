# Uploaded Event Banner Images

This folder stores all uploaded event banner images.

**Location:** `projectbackend/uploads/images/`

**Features:**

- Automatic file naming with timestamps
- 5MB file size limit
- Accepts: JPG, JPEG, PNG, GIF, WebP
- Images accessible via: `http://localhost:5000/uploads/images/filename.jpg`

**Note:** This folder is automatically created by the server if it doesn't exist.
